Hackintosh Log #1

August 29, 2021
Edited: September 29, 2021

Hello, welcome to my first log of my experience doing a Hackintosh. I've been using macOS Mojave for the past 20 days and learned alot about macOS and Hackintosh in general.

A brick wall

For awhile i've been wanting to do a Hackintosh on my main system. Though I hit a brick wall, my iGPU is a Intel HD 2500. Which is not supported on macOS.

The HD 2500 on the other hand only has partial support in Mojave for quick sync features as hardware acceleration is unsupported so you will need a compatible dGPU for display purposes.

But didn't stop me from doing a Hackintosh. I bought a used GT 630 from eBay, which is still supported on macOS. Now, I was able to start doing the Hackintosh.

More Problems

I was originally going to make an OpenCore Configuration. But after macOS Mojave finished the installation, there was no option for "macOS Installer", which you'd needed to finish the installation.

I eventually gave up after tinkering and tweaking config.plist and to no avail, and used a premade one on GitHub . Which to my surprise it worked, after some time I was on Setup Assistant. I completed the setup and was glad to see the Mojave wallpaper and desktop.


Screenshot of About This Mac.

First Thoughts

After copying the EFI folder to my SSD and tweaking OpenCore, I started installing programs and tweaking macOS to my liking.

Almost everything works on the Hackintosh, unfortunately the only thing that didn't work is my Wi-Fi card. Which is an Qualcomm Athernos AR93xx. Even after getting help and hair-pulling trying to get it to work, I never got it to work. So I had to resort to using Ethernet.


Current macOS Mojave setup.

Current Issues

Currently, not everything works right now. The issues currently are:

  • Some USB ports don't work
  • "EFI" entry shows on Boot Picker despite nothing but the OC folder
  • Incompatible Wi-Fi card
  • Sleeping is iffy
  • iMessage/FaceTime not working

Currently trying my best on fixing these issues, but aren't really a big issue for me.

Final Thoughts

It has been over 20 days since I started this, Overall this Hackintosh endeavour went pretty smoothly for me. My previous Hackintosh projects were a pita because of sound/graphics issues. I will post a update when I make a update to OpenCore or do more tweaking to macOS.
